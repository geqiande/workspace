# 钉钉目录树

```cmd

${root}/ 根目录
    css/
      .../ css文件
    fonts/
      .../ 字体文件
    icon/
      .../ 图标
    images/
      .../ 静态图片
    js/
      skin/
        default/
          .../ 提示框配置文件
      .../ js文件
    template/
      .../ 模板文件(切图需要，可删)
    project/	
     .../项目
    staff/		
     .../员工
    client/		
     .../客户
    message/	
     .../消息
    index.html 入口文件

```
