# 钉钉目录树

```cmd

${root}/ 根目录
    css/
      .../ css文件
      workbench.css 工作台
      index.css 首页
      public.css 公共
    fonts/
      .../ 字体文件
    icon/
      .../ 图标
    images/
      .../ 静态图片
    js/
      skin/
        default/
          .../ 提示框配置文件
      .../ js文件
    template/
      .../ 模板文件(切图需要，可删)
    project/	
     .../项目
     projectdetails.html 项目详情
     projectinformation.html 项目资料
     projectlist.html 项目列表
     projectdivisionoflabor.html 项目分工
     testdetails.html 测试详情
     projectstructure.html 项目列表
     projecttesting.html 项目测试
     listofcomplaints.html 申诉列表
     taskreview.html 任务审核
    staff/		
     .../员工
     workbench.html 工作台
     employeelist.html 员工列表
     authoritymanagement.html 权限管理
    client/		
     .../客户
     withdetails.html 跟单详情
     customerdetails.html 客户详情
     customerlist.html 客户列表
     
    message/	
     .../消息
     mymessageforall.html 群聊
     mymessageforone.html 单聊
     messagelist.html 消息列表
     
    schedule/
     .../日程
     logquery.html 日志查询
     worklog.html 工作日志
     schedule.html 日程管理
     worklogmy.html 我收到的日志
     worklogmysend.html 我发出的日志
     worklogmylove.html 我收藏的日志
     logdetails.html 日志详情
    index.html 入口文件

```
